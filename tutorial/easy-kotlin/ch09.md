## 9.2 协程的基本操作

### 9.2.2 简单协程示例

```kotlin
fun firstCoroutineDemo0() {
    launch(CommonPool) {
        delay(3000L, TimeUnit.MILLISECONDS)
        println("Hello,")
    }
    println("World!")
    Thread.sleep(5000L)
}

fun main(args: Array<String>) {
    firstCoroutineDemo0()
}
```

输出结果

```bash
World!
Hello,
```
