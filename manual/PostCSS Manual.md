## [Installation & Setup](https://github.com/postcss/postcss#usage)

### [Webpack](https://github.com/postcss/postcss#webpack)

`webpack.config.js`
```javascript
module.exports = {
    module: {
        rules: [
            {
                test: /\.css$/,
                exclude: /node_modules/,
                use: [
                    {
                        loader: 'style-loader',
                    },
                    {
                        loader: 'css-loader',
                        options: {
                            importLoaders: 1,
                        }
                    },
                    {
                        loader: 'postcss-loader'
                    }
                ]
            }
        ]
    }
}
```

`postcss.config.js`
```javascript
module.exports = {
    plugins: [
        require('precss'),
        require('autoprefixer')
    ]
}
```
