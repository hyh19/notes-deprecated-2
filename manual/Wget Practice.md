# Wget Practice


```
$ wget -h
Usage: wget [OPTION]... [URL]...

Startup:
  -h,  --help                      print this help

Logging and input file:
  -q,  --quiet                     quiet (no output)

Download:
  -O,  --output-document=FILE      write documents to FILE

Directories:
  -P,  --directory-prefix=PREFIX   save files to PREFIX/..
```
