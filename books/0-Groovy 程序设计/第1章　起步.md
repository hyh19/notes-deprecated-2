## 1.5　在命令行中运行Groovy

```bash
$ cat Hello.groovy 
println "Hello Groovy!"
$ groovy Hello.groovy 
Hello Groovy!
```

在命令行中直接运行一些语句
```bash
$ groovy -e "println 'hello'"
hello
```
