# 习题 10 那是什么

```
>>> print "I am 6'2\" tall." # 将字符串中的双引号转义
I am 6'2" tall.
>>> print 'I am 6\'2" tall.' # 将字符串中的单引号转义
I am 6'2" tall.
```

```python
tabby_cat = "\tI'm tabbed in."
persian_cat = "I'm split\non a line."
backslash_cat = "I'm \\ a \\ cat."

fat_cat = """
I'll do a list:
\t* Cat food
\t* Fishies
\t* Catnip\n\t* Grass
"""

print tabby_cat
print persian_cat
print backslash_cat
print fat_cat
```

输出结果：
```
    I'm tabbed in.
I'm split
on a line.
I'm \ a \ cat.

I'll do a list:
    * Cat food
    * Fishies
    * Catnip
    * Grass
```

使用 `'''` 处理多行字符串
```python
fat_cat = '''
I'll do a list:
\t* Cat food
\t* Fishies
\t* Catnip\n\t* Grass
'''

print fat_cat
```

输出结果：
```
I'll do a list:
    * Cat food
    * Fishies
    * Catnip
    * Grass
```
