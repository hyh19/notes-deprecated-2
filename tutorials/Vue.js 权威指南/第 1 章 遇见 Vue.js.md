# 第 1 章 遇见 Vue.js

```html
<div id="didi-navigator">
    <ul>
        <li v-for="tab in tabs">{{ tab.text }}</li>
    </ul>
</div>
```

```javascript
new Vue({
	el:'#didi-navigator',
    data: {
    	tabs: [
        	{ text: '巴士' },
            { text: '快车' },
            { text: '专车' },
            { text: '顺风车' },
            { text: '出租车' },
            { text: '代驾' }
        ]
    }
})
```
